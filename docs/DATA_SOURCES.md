# Data Sources

This extension uses open-source and public data sources.

## crt.sh

- Certificate Transparency search
- Passive discovery of certificate names
- https://crt.sh

## dns.bufferover.run

- Passive DNS index
- Returns historical DNS names
- https://dns.bufferover.run

## dns.google

- DNS-over-HTTPS
- Used for validation
- https://dns.google
