# SEO Usage Summary

Short checklist for SEO teams.

## Quick Checklist

- Run passive scan.
- Export CSV.
- Filter high value subdomains.
- Validate DNS and HTTP.
- Audit indexability and content.
- Consolidate or redirect where needed.

## Common Patterns

- staging, dev, test: usually noindex or blocked.
- blog, shop, store: high value review targets.
- old, legacy, archive: consolidation candidates.

## Suggested Exports

- CSV for spreadsheet review.
- JSON for automation and pipelines.
